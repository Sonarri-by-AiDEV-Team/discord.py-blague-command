If your host your code  on a Pterodactyl Panel (Pterodactyl Host) : 

Add on file requirements.txt -> blagues_api
Add at the very beginning of the file app.py or bot.py -> from blagues_api import BlaguesAPI

I your host your code on a Machine with Virtual Studio Code :

Install blagues_api with : pip install blagues_api
Add at the very beginning of the file app.py or bot.py -> from blagues_api import BlaguesAPI

AND ITS READY !
