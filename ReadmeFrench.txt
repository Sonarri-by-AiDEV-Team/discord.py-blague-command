Si vous hébergez votre bot sur un panel pterodactyl :

Ajouter dans le fichier requirements.txt -> blagues_api
Ajoutez au tout début du fichier (bot.py/app.py -> from blagues_api import BlaguesAPI

Si vous hébergez votre bot sur une machine avec Virtual Studio Code :

Installez blagues_api avec -> pip install blagues_api
Ajoutez au tout début du code (bot.py/app.py) -> from blagues_api import BlaguesAPI
